## pygame-rocket

![preview](./media/screenshot.png)


#### Dependencies

`pip install pygame>=2.1.3`


#### Run

`python game.py`


#### Controlls

Thrust: spacebar

Turn: arrows


#### Improvement suggestions

- Proper start and game over screens
- Pixel-perfect collision detection with [masks](https://github.com/Rabbid76/PyGameExamplesAndAnswers/blob/master/documentation/pygame/pygame_mask.md)
- Rocket acceleration and deacceleration
- Meteor collisions break meteors into smaller meteors
- Rocket can shoot meteors to break them
- Multiplayer and [controller support](https://www.pygame.org/docs/ref/joystick.html)
